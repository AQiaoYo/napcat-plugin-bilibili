插件仪表盘（模板）

这是一个通用的插件仪表盘模板。请在开发新插件时替换或扩展此页面。

快速提示

  • 路由前缀由后端 ROUTE_PREFIX 控制，默认：/plugin-template
  • 插件可通过 REST 接口读取/保存配置：GET /plugin-template/config
  • 通过脚本 /plugin-template/static/plugin-info.js 可以获取注入的插件名

示例：使用 authFetch

            
            async function authFetch(path, opts = {}) {
                const token = new URLSearchParams(location.search).get('webui_token') || '';
                const url = `/plugin-template${path}?webui_token=${token}`;
                const res = await fetch(url, opts);
                return res.json();
                }
